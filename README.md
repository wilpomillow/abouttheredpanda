# Red Panda Field Guide (Polished)

A visually-stunning, highly animated **one-page explainer** about red pandas.
Built with **Next.js** for **Vercel**, plus **Tailwind**, **Framer Motion**, and **Anime.js**.

## Run locally
```bash
npm install
npm run dev
```

Open: http://localhost:3000
